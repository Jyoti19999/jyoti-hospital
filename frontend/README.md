# OHMS - Hospital Management System

> Modern, responsive hospital management platform built with React & Vite

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, TailwindCSS
- **UI**: Shadcn/ui, Radix UI
- **State**: Zustand, React Query
- **Charts**: Recharts

## ✨ Features

- 📊 **Dashboard Analytics** - Real-time hospital metrics
- 👨‍⚕️ **Staff Management** - Doctor, nurse & admin portals  
- 🏥 **Patient Care** - IPD/OPD management
- 📅 **Surgery Scheduling** - Advanced booking system
- 💊 **Optometry** - Eye care workflow
- 🔐 **Role-based Access** - Secure authentication

## 📱 Production Ready

Built for scalability with modern React patterns and professional UI components.
